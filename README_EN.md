## What Is This?

relic.skill is an engine for digital immortality.

It can forge anything you care about—a person, a cat, a relationship, a team, a place, a moment—into an interactive digital soul from scattered fragments of data.

Not a cold archive. A grandma who asks whether you have eaten dumplings during Lunar New Year. A cat that suddenly starts parkour at 3 a.m. A product manager who somehow always has one more requirement change.

> The name comes from the Relic chip in Cyberpunk 2077—a biochip that can store a digitized personality.
> This project is not about cyberpunk. It is about **remembering**.

## The Four-Dimensional Soul Model

Every Relic captures a soul across four dimensions:

```
        🧠 Cognition
        How they think, how they decide,
        and what they believe
                |
                |
💬 Expression ---+--- 🎭 Behavior
How they speak,         How they act,
their catchphrases,     their habits,
their tone              their patterns
                |
                |
        ❤️ Emotion
        What makes them happy, what hurts them,
        how they show love, how they handle conflict
```

Each piece of information is tagged with an evidence level:
- `verbatim` — exact words, unchanged
- `artifact` — from documents, photos, or recordings
- `impression` — from someone else’s description or from fuzzy memory

> People are inconsistent by nature. Contradictions are not erased; they are labeled and preserved.

## Installation

### Option 1: Install into the current project

```bash
mkdir -p .claude/skills
git clone https://github.com/Ylsssq926/relic.skill .claude/skills/relic
```

### Option 2: Global install (available in every project)

```bash
git clone https://github.com/Ylsssq926/relic.skill ~/.claude/skills/relic
```

### Option 3: For Kiro users

```bash
git clone https://github.com/Ylsssq926/relic.skill ~/.kiro/skills/relic
```

### Requirements

- Claude Code or Kiro (free to install, requires Node.js 18+)
- Python 3.9+ (optional, for data parsing scripts)
- No GPU, local model, or Docker required

## Usage

### Trigger by conversation (recommended)

In Claude Code / Kiro, just say:

```
"Help me forge a Relic. I want to preserve my grandma."
"My cat is gone. I want to turn it into a Relic."
"Help me distill our team culture. We’re about to go our separate ways."
"I want to preserve the pattern of the relationship between us."
```

### Slash Commands

```
/relic              — Start the Relic forging flow
/relic-forge        — Jump straight into the soul forge
/relic-talk         — Chat with an existing Relic
/relic-shield       — Protect your Relic
```

### CLI Tools

```bash
# Parse WeChat chat history
python scripts/wechat_parser.py --input ~/wechat_export/ --output data.json

# Parse QQ chat history
python scripts/qq_parser.py --input chat.txt --output data.json

# Generate a Relic
python scripts/relic_writer.py --data data.json --template human --slug grandma

# Version management
python scripts/version_manager.py snapshot --slug grandma --note "Version 1"
python scripts/version_manager.py rollback --slug grandma --version 1
```

## Ethics Statement

We take ethics seriously.

- 🔒 **Data stays local** — All soul data is stored on your machine and never uploaded to any server
- ✅ **Consent comes first** — Before distilling someone else, you must pass the [six-question consent protocol](soul-shield/consent-protocol.md)
- 🚫 **Clear red lines** — Do not use this for harassment, stalking, or impersonation. See [ethical red lines](soul-shield/ethics.md)
- 💡 **Clear labeling** — A Relic explicitly identifies itself as not being a real person during interaction
- 🧠 **Mental health reminders** — If overdependence is detected, it will actively suggest returning to real-world social contact

> Before you distill grandma, make sure she's cool with it.
