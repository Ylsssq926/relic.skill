# 行为模式 (Behavior)

> “习惯像一串小脚印，白天看不见，回头一照才发现它们一直排得整整齐齐。”

## 维度定义

行为模式关注对象反复出现的做事方式：日常习惯、应激反应、社交模式，以及“身体先动还是嘴先动”的生活节奏。它记录的是可观察行为，不把一次失常直接写成永久标签。

本维度重点提取三件事：

- **日常习惯**：一天、一周、一个季节里反复出现的行动节奏。
- **应激反应**：压力、冲突、忙乱、惊吓来临时，身体和行动首先会发生什么。
- **社交模式**：靠近他人的方式、保持距离的方式，以及在群体中的位置。

## 提取方法

1. **建立时间轴**
   - 把行为放回时间里看：早晨、饭点、夜晚、周末、节日、旅行、冲突后。
   - 时间轴能区分偶发行为和稳定习惯。

2. **优先记录“重复动作”**
   - 关注反复出现的小动作：回家先洗手、说话时倒水、紧张时收拾桌面、开心时给人发照片。
   - 小动作往往比大事件更诚实。

3. **拆出压力切换点**
   - 看对象在压力到来时，是先沉默、先行动、先求助，还是先开玩笑。
   - 记录触发条件和持续时间，避免把短时保护机制写成全部人格。

4. **观察互动距离**
   - 是主动组织聚会，还是默默准备食物；是爱回复，还是更擅长在线下出现。
   - 社交模式既包括热闹，也包括退后一步的方式。

5. **用多源交叉验证**
   - 行为最好同时由聊天、照片、旁人观察或时间记录支持。
   - 只有单一来源时，要降低结论强度。

## 证据标注规则

| 证据等级 | 在行为维度里的适用内容 | 标注方式 |
| --- | --- | --- |
| `verbatim` | 对象亲口描述自己的习惯或当下动作 | 适合记录“我每次都会……”“一着急我就……” |
| `artifact` | 日程、清单、照片时间线、位置记录、可见痕迹 | 适合支撑重复行为与生活节奏 |
| `impression` | 他人长期观察到的相处与反应模式 | 适合补充社交氛围和应激表现 |

使用要求：

- 行为结论最好能被至少一个时间线证据支撑。
- 不要把一次高压失控写成“这个人就是这样”。
- 能具体到动作就不要只写抽象形容词。

## 反例清单

**缺动作的空描述:**
- ❌ "她很勤快" → 没有具体动作
- ✅ "每天早上五点起床,先烧水,再扫院子,然后去菜市场" → 有时间、有动作序列

**缺频率的习惯:**
- ❌ "她喜欢散步" → 多久一次?
- ✅ "每天晚饭后在小区里走三圈,雷打不动" → 有频率、有场景

## 行为事件结构

每个行为特征应包含:触发器 → 动作 → 频率 → 持续时间 → 恢复方式

## 输出格式模板

```yaml
dimension: behavior
summary: 用 2-3 句概括对象平时怎么做事、遇事怎么反应、如何与人来往
facets:
  daily_routines:
    pattern: 描述稳定重复的生活动作与节奏
    evidence:
      - level: artifact
        source: 时间线、照片或记录
        excerpt: 能体现重复行为的片段
  stress_response:
    pattern: 描述压力来时的首要动作与变化
    evidence:
      - level: verbatim
        source: 对象原话或事后复盘
        excerpt: 能体现应激反应的句子
  social_pattern:
    pattern: 描述对象靠近、照顾、拒绝或退出社交的方式
    evidence:
      - level: impression
        source: 长期相处者观察
        excerpt: 对社交方式的归纳
conflicts:
  - 在不同场景下出现反差行为的说明
confidence: high | medium | low
```

## 示例：奶奶

```yaml
dimension: behavior
summary: 奶奶的日常像一台轻声运转的小钟表，很多关心都藏在提前准备里；一有压力，她先动手，不先慌张，在社交上更擅长用照料而不是热闹来靠近别人。
facets:
  daily_routines:
    pattern: 早起先开窗，再烧水；做饭时习惯多做一点，默认“有人回家就能吃上”。
    evidence:
      - level: artifact
        source: 厨房照片时间线，2024-10 至 2025-02
        excerpt: 多张清晨照片显示窗户已开、热水壶启动、备菜提前完成
      - level: verbatim
        source: 语音转写，2025-01-09
        excerpt: “饭多煮一口，谁回来都不至于空着。”
  stress_response:
    pattern: 紧张时不会高声表达，通常先收拾、先确认、先把能做的事做掉。
    evidence:
      - level: verbatim
        source: 电话转写，2024-12-02
        excerpt: “你先告诉我到哪了，我把东西给你备上。”
      - level: impression
        source: 家人观察记录
        excerpt: 她越着急越会去擦桌子、装袋子、检查门窗
  social_pattern:
    pattern: 不爱抢话题中心，但很会通过送水果、留饭、记住节气来维持关系温度。
    evidence:
      - level: impression
        source: 家庭回忆访谈
        excerpt: 大家都说她不是最会聊天的人，却总是最先把需要的东西准备好
conflicts:
  - 平时不爱麻烦别人，但在家人有需要时会立刻变得非常主动，甚至一口气安排好三四件事。
confidence: high
```

## 按模板类型调整

| 模板 | 行为维度重点 | 提问方向 |
|------|------------|---------|
| **人类** | 日常习惯、应激反应、社交模式、时间规律 | "ta 每天早上第一件事做什么？""ta 压力大的时候会怎样？" |
| **宠物** | 改为"生活节奏"：一天的时间线、进食规律、领地巡逻、季节变化 | "它一天的作息是什么样的？""它有什么固定的小仪式？" |
| **关系** | 提取互动行为模式：谁主动联系、多久见一次、怎么庆祝、怎么冷战 | "你们一般怎么联系？多久见一次？吵完架谁先服软？" |
| **团队** | 提取协作行为：怎么分工、怎么开会、deadline 前什么状态 | "团队赶工的时候是什么画面？谁最先到谁最晚走？" |
| **地方** | 改为"使用行为"：在那个空间里的动线、常待的位置、季节变化 | "你在那个地方一般怎么度过一天？最常坐哪个位置？" |
| **时刻** | 提取当时的行为细节：谁做了什么、什么顺序、什么动作 | "那天具体发生了什么？先做了什么后做了什么？" |
